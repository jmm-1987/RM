Información sobre Green-API
¿Qué es Green-API?

Green-API es un servicio que permite enviar mensajes de WhatsApp de forma automatizada a través de su API REST.

Configuración necesaria:
  • API URL: La URL de tu instancia (ej: https://7107.api.green-api.com)
  • API Token: El token de autenticación de tu instancia
Pasos para configurar:
  1. Accede a tu panel de Green-API
  2. Copia la URL de tu instancia
  3. Copia el token de autenticación
  4. Configura estos datos en el sistema
  5. Realiza una prueba de envío
Importante: Asegúrate de que tu instancia de Green-API esté autorizada y funcionando correctamente.
